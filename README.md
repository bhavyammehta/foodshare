# FoodShare

## Project Overview
FoodShare is a web-based platform that helps reduce food waste by connecting food donors with people and organizations in need.

## Features
- User Registration and Login
- Food Donation Management
- Food Request System
- Donor and Receiver Dashboard
- Secure Data Management

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## Installation

1. Clone the repository:
   
   git clone https://github.com/bhavyammehta/foodshare.git
  

2. Navigate to the project directory:
   cd foodshare

3. Install dependencies:
   pip install -r requirements.txt

4. Run migrations:
   python manage.py migrate
5. Start the server:
   python manage.py runserver
