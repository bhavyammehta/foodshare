 FoodShare

FoodShare is a Django-based web application designed to reduce food wastage by providing a platform where users can donate and manage surplus food efficiently.

---

 Features

* User Registration & Login
* Add and Manage Food Donations
* Track Available Food Items
* Simple and User-Friendly Interface

---

Tech Stack

Backend: Django (Python)
Database: SQLite
Frontend: HTML, CSS

Objective

The main goal of this project is to minimize food wastage and help people in need by connecting food donors with recipients through a centralized platform.

---

 How to Run the Project

bash
Clone the repository
git clone https://github.com/bhavyammehta/foodshare.git

Go to project folder
cd foodshare

Run server
python manage.py runserver
